# Práctica 1 - Integración de API REST con React

## Descripción

Aplicación desarrollada con **React + TypeScript** que consume datos reales de la API pública **SWAPI (Star Wars API)**.

La aplicación:

- Muestra personajes en formato tarjeta.
- Gestiona estado de carga y errores.
- Implementa paginación mediante la propiedad `next` devuelta por la API.
- Acumula los personajes cargados sin reemplazar los anteriores.

---

## Tecnologías utilizadas

- React
- TypeScript
- Axios
- Vite
